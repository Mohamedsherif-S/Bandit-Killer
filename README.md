# OverTheWire: Bandit Writeups

My personal notes and solutions for the Bandit wargame — a beginner-friendly Linux & security challenge.

> **Note:** No passwords are shared here. Only concepts, commands, and lessons learned.

## Skills Covered
- Linux command line basics
- File permissions & SUID binaries
- Networking (nc, openssl, nmap, SSH)
- Git (log, branches, tags, push)
- Shell escaping techniques
- Cryptography basics (base64, rot13, gzip, tar)
